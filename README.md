## Tecnologias utilizadas

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML e CSS
- Typescript
- React
- Tailwindcss

## Instalando localmente

Siga o passo a passo descrito abaixo:

```bash
  git clone https://github.com/Cauatn/scasys.git
  cd my-project
  npm install
```
